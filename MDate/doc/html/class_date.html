Ausführliche Beschreibung

MDate.h

Diese Headerdatei sorgt für die Implementierung einer Klasse die für Datum- und Zeiterfassung zuständig ist. Diese Klasse bietet verschiedene Funktionen um ein Datum zu erfassen, darzustellen und wiederzugeben.

◆ Get_Date()

std::string Date::Get_Date ( )

Gibt das Datum im Format 'YYYY-MM-DDTHH:mm:ss' zurück. YYYY : Jahr MM : Monat (zweistellige) DD : Tag im Monat (zweistellige) HH : Stunde (zweistellige) mm : Minute (zweistellige) ss : Sekunde (zweistellig)

◆ Get_Month()

int Date::Get_Month ( )

Gibt den Monat zurück, dabei wird von 0 an gezählt. Also gilt 0<=x<=11. 0 : Januar 1 : Februar 2 : März 3 : April 4 : Mai 5 : Juni 6 : Juli 7 : August 8 : September 9 : Oktober 10 : November 11 : Dezember

◆ Get_Weekday()

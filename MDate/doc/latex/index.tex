\hypertarget{index_intro_sec}{}\doxysection{Beschreibung}\label{index_intro_sec}
Die \char`\"{}\+Date\char`\"{} Klasse bietet verschiedene Funktionen für Erstellung und Verarbeitung eines Datums. Damit wird die Handhabung mit einem Datum erleichtert. Man kann ein Datum auf verschiedene Weisen erstellen, z.\+B. aus der Systemuhr des Computers oder auch einfach mit ganzzahligen Parametern. Es ist darauf zu achten, dass kein Datum vor 1970 verwendet wird, es könnte nämlich zu Fehlern beim Kompilieren führen. Die Klasse ist unter \char`\"{}\+Klassen\char`\"{} und dann \char`\"{}\+Date\char`\"{} verfügbar.\hypertarget{index_install_sec}{}\doxysection{Installation}\label{index_install_sec}
Einfach die Datei \char`\"{}\+MDate.\+h\char`\"{} in das Include-\/\+Verzeichnis hineinkopieren und die Datei \char`\"{}\+Date.\+cpp\char`\"{} in das Quellverzeichnis hineinkopieren. Damit der Code auch funktioniert, wird die Boost Bibliothek benötigt.
